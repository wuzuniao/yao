# Checklist

以下检查点用于验证 `spec.md` 是否完整、准确地梳理了记录页（record.vue）的打卡记录显示逻辑（以用户本次给出的新要求为准）。逐项核对，全部通过即视为梳理完成。

## 现有逻辑梳理（对比基准）
- [x] 已梳理现有小绿点标注依据（基于 `checkin_records` 表的 `listMonthCheckins` 返回值）
- [x] 已梳理现有每日打卡明细展示规则（每记录一行 + 孤儿记录单独展示）
- [x] 已梳理现有明细行数据结构（`{ notification_time, checked, actual_time }`）
- [x] 已梳理现有匹配区间算法（按相邻中点划分，全天 0:00-24:00 无留白）
- [x] 已梳理现有数据加载触发时机（`onShow` 加载月度记录 + 自动选中今天）
